# Raspberry PI - Set Wifi: iOS / Android App

## Installing

```bash
$ yarn install
$ cd ios && pod install
```

## Running
```bash
# IOS
$ yarn run ios
# ANDROID
$ yarn run android
```

## Assembling Release
```bash
# ANDROID
$ cd android && ./gradlew assembleRelease
```

## Editing configurations
Open the file `config.js`
